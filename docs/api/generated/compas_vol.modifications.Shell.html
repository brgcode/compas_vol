Shell

class compas_vol.modifications.Shell(obj, thickness=1.0, side=0.0)

Bases: object

A shell object converts a solid volumetric object into a constant thickness boundary volume.

Parameters

  • obj (compas_vol) – The input object.

  • thickness (float) – The thickness of the shell.

  • side (float) – Side factor. For s = 1.0, the shell will be offset to the inside. For s = 0.5, the shell will be offset half/half on either side of the original surface. For s = 0.0, the shell will be offset to the outside.
